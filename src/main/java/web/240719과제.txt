[ 과제 주제 ] 인사 관리 페이지
	- 부서 테이블 {부서번호, 부서명, 대표전화}
	- 인사테이블 {인사번호, 사원명, 연락처, 직급, 부서번호FK}

	- DB명 : HRDsystem, 테이블명 : 임의

[ 기능 구현 ]
	1. 부서 등록, 부서 전체 출력, 부서 수정(부서명, 대표전화), 부서삭제 구현
	2. 인사 등록, 인사 전체 출력, 인사 수정(사원명, 연락처, 직급), 인사삭제 구현
		- 인사 등록시 부서 목록에 존재하는 부서번호를 참조항여 인사정보에 등록 하시오
